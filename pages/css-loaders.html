System Status Indicators

Minimalist CSS Loaders.
Lightweight, Scalable, No JavaScript Required.


01. Data Stream

データの送受信や音声波形を表現するバーアニメーション。

CSS Keyframes
.bar {
  animation: dataStream 1s ease-in-out infinite;
}
.bar:nth-child(1) { animation-delay: 0.0s; }
.bar:nth-child(2) { animation-delay: 0.1s; }
/* ... */

02. Reactor Ring

多重構造のリングが異なる速度で回転する待機インジケータ。

Border Spin
.arc {
  border: 2px solid transparent;
  border-top-color: #fff;
  border-radius: 50%;
  animation: spin 1.5s cubic-bezier(0.68, -0.55, 0.265, 1.55) infinite;
}

03. System Scan

リニアグラデーションを使用した走査線エフェクト。

Gradient Animation
.scan-line {
  background: linear-gradient(90deg, transparent, #fff, transparent);
  transform: translateX(-100%);
  animation: scanning 1.5s ease-in-out infinite;
}

04. Neural Grid

3x3のグリッドパターンがランダム風に明滅。

Grid & Delay
.loader-grid { display: grid; grid-template-columns: repeat(3, 1fr); }
.dot { animation: pulse 1.2s infinite; }
/* 各dotに異なるanimation-delayを設定 */

05. Kinetic Cube

3Dトランスフォームを使用した、折り畳まれるような正方形。

3D Rotation
@keyframes kinetic {
  0% { transform: perspective(120px) rotateX(0deg); }
  50% { transform: perspective(120px) rotateX(-180deg); background: #555; }
  100% { transform: perspective(120px) rotateX(-180deg) rotateY(-180deg); }
}
