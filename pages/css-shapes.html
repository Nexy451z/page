03. Gyroscope Model

3つのリングが異なる軸で回転し、球体のような軌道を描く。

Multi-Axis Rotation 
/* 異なる軸で回転させるアニメーション */
@keyframes rotate1 { to { transform: rotateX(360deg) rotateY(360deg); } }
@keyframes rotate2 { to { transform: rotateX(360deg) rotateY(-360deg); } }
.ring-1 { animation: rotate1 4s infinite linear; }
.ring-2 { animation: rotate2 5s infinite linear; }

04. Hyper Tesseract

入れ子になったキューブが互いに逆回転

Nested 3D Transforms 
.outer-cube {
  animation: spin 10s linear infinite;
}
.inner-cube {
  /* 親の回転の中でさらに逆回転させる */
  animation: spin 4s linear infinite reverse;
  /* 透過度とサイズを変えて奥行きを強調 */
}

7. The Singularity

重力。事象の地平面へ吸い込まれる物質の表現。

Gravity Well Effect
/* 事象の地平面へ加速して落ち込むアニメーション */
@keyframes suckMatter {
  0% { transform: scale(1.5) rotate(0deg); opacity: 0; }
  30% { opacity: 0.6; }
  100% { transform: scale(0) rotate(360deg); opacity: 0; }
}
.infall {
  animation: suckMatter 1.5s cubic-bezier(0.55, 0.055, 0.675, 0.19) infinite;
}
